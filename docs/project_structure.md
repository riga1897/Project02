
# Структура проекта Project02

## Общее описание
Приложение для поиска вакансий с поддержкой множественных источников данных, кэширования и удобного пользовательского интерфейса.

## Ключевые особенности архитектуры

### 1. Модульная структура
- Каждый модуль имеет четко определенную ответственность
- Использование абстрактных базовых классов для унификации интерфейсов
- Разделение логики на слои: API, модели, UI, утилиты

### 2. Расширяемость
- Легко добавить новые источники вакансий через наследование от BaseJobAPI
- Модульная система парсеров для различных форматов данных
- Гибкая система конфигурации

### 3. Производительность
- Многоуровневое кэширование (память + файлы)
- Пагинация для работы с большими объемами данных
- Асинхронная обработка запросов с визуализацией прогресса

### 4. Надежность
- Обширное покрытие тестами
- Валидация данных на всех уровнях
- Обработка ошибок и восстановление после сбоев
